High Availibility on AKS

Kubernetes itself is known for High Availibility and resiliency. There is Detailed Decumentation on MS Website. The Image below is taken from MS documentaion

Zone Redundant Nodepool

On production setup, the Nodepools will be zone redundant. So that each replica of the pod will be created on a new Availibility Zone

Geo Redundant Nodepool

As 2nd level of Fault Tolerance, we can make another Nodepool in another Geo Location.

Geo Redundant AKS Cluster

As 2nd level of Fault Tolerance, we can make a replica of the whole cluster in another geo location and use it on demand.
